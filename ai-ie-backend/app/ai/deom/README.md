# deom

This folder keeps demo and one-off scripts that should not live inside
`app/ai`.

The name follows the user's requested folder spelling. In a formal project,
you may rename it to `demo` or `examples`.

## Files

- `minimal_chat_demo.py`: minimal SQLite + LlamaIndex chat memory demo.
- `lightrag_ingest_demo.py`: markdown ingestion demo for a LightRAG instance.
- `lightrag_chat_demo.py`: LightRAG chat demo showing how to plug the backend into `ChatAI`.
