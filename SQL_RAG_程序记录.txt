git switch main
解释：切回主分支。

git pull --ff-only origin main
解释：从 GitHub 拉最新 main，只允许快进合并，避免乱合并。

git switch -c feature/你的功能名
例子：

git switch -c feature/sql-rag-qdrant-sync
解释：从当前 main 新建一个功能分支。

git push -u origin feature/sql-rag-qdrant-sync
